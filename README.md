# STEP Academy Landing Page

صفحة هبوط لموقع اختبارات STEP مصممة باستخدام Next.js و Tailwind CSS

## المميزات
- تصميم مستوحى من Google
- متجاوب مع جميع الأجهزة
- أنيميشن وتفاعلات سلسة
- دعم اللغة العربية

## التشغيل
```bash
npm install
npm run dev
```

## التقنيات المستخدمة
- Next.js 15
- Tailwind CSS
- Lucide React Icons
- TypeScript
